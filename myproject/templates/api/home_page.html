IOTA-Based Distributed Computing

Iotagrid is iota-based distributed computing system that compute complex task by distributing it in the peer to peet network. First application of the system is computing f(x,y) = x^2 + y^2. We will add more tasks in the future. In this task user insert its two inputs and then system distribute inputs amoung iota nodes. When nodes calculate their outputs, they sended it to the network. Finally, first node calculate the final answer.
