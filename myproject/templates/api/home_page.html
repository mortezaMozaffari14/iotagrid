IOTA-Based Distributed Computing
    Distribute task Private network info Create new task All tasks

Iotagrid is an IOTA-based distributed computing system that computes complex tasks by distributing it in the IOTA network. First application(Demo) of the system is computing f(x,y) = x^2 + y^2. We will add more tasks in the future. In this task, user insert its two inputs and then system distributes inputs among IOTA nodes. When nodes calculate their outputs, they send it to the network. Finally, the first node calculates the final answer. Payment will be added to the system in the near future.
